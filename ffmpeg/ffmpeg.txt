1.ffmpeg安装
->下载最新源码包并解压
wget http://ffmpeg.org/releases/ffmpeg-3.1.3.tar.bz2
tar xvjf ffmpeg-3.1.3.tar.bz2

->安装ffmpeg
cd ffmpeg-3.1.3
./configure
此时可能会报错,提示yasm没有安装

->安装yasm
如果报错了就安装yasm，否则不用安装
apt-get install yasm

->继续按照ffmpeg
./configure
make
make install

->ffmpeg
输入ffmpeg命令，打印下相关信息，表示安装成功
